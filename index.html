PORTFOLIO

TaskMaster is a mobile-first web application that allows people to easily create and share task lists with each other. Users can create accounts, then task lists, and share the lists with other registered users. Users can check off tasks as they complete them, which then updates on the application in real-time. TaskMaster was a collaboration project by myself, Jeff Weinreich (Back End Developer) and Ian Bolling (Front End Developer).

Technologies used: HTML5, CSS, SCSS, JavaScript, Backbone, React, C#, .NET
